# 个人财产管理工具 - 产品需求文档

## 1. 产品概述

一款面向个人用户的财产管理工具，帮助用户收集、录入、整理和统计个人资产与负债情况。通过直观的仪表盘和分类管理，让用户清晰掌握自身财务状况。

- 主要用途：个人资产全景视图、分类管理、统计分析
- 目标用户：有资产管理需求的个人用户
- 核心价值：一站式掌握个人财务健康状况

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 个人用户 | 无需注册（本地单用户） | 全部功能：录入、查看、统计、导出数据 |

### 2.2 功能模块

1. **仪表盘总览**：资产全景、净资产概览、分类占比、趋势图表
2. **银行存款管理**：活期/定期存款录入、编辑、删除、汇总
3. **证券投资管理**：股票、债券等证券资产录入与市值跟踪
4. **理财和基金管理**：理财产品、公募基金、私募基金管理
5. **贷款和信用卡管理**：房贷、车贷、消费贷、信用卡欠款管理
6. **统计分析**：资产结构分析、负债率计算、净资产趋势、分类对比

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 仪表盘 | 资产概览卡片 | 展示总资产、总负债、净资产、负债率四个核心指标 |
| 仪表盘 | 资产分布饼图 | 按类别展示资产占比（银行存款、证券、理财基金） |
| 仪表盘 | 负债分布饼图 | 按类别展示负债占比（贷款、信用卡） |
| 仪表盘 | 近期变动列表 | 展示最近添加或修改的资产/负债记录 |
| 资产录入 | 分类录入表单 | 按四大类别分别提供录入表单，支持动态添加字段 |
| 资产录入 | 资产列表 | 分类展示所有已录入资产，支持编辑和删除 |
| 统计分析 | 资产结构分析 | 饼图/柱状图展示各类资产占比 |
| 统计分析 | 负债结构分析 | 饼图/柱状图展示各类负债占比 |
| 统计分析 | 净资产趋势 | 折线图展示净资产变化趋势（按时间维度） |
| 统计分析 | 关键指标 | 储蓄率、负债率、资产流动性比率等 |

## 3. 核心流程

用户打开应用后，首先进入仪表盘查看资产全景。通过导航进入各分类管理页面，录入新的资产或负债信息。系统自动计算汇总数据并更新仪表盘。用户可在统计分析页面查看详细的资产结构和趋势分析。

```mermaid
    E --> J["查看/编辑/删除记录"]

    K --> B
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深墨绿（#1a3a3a）+ 金色（#c9a96e）— 传达稳重、财富、专业感
- **辅助色**：暖灰（#f5f0eb）背景、深灰（#2d2d2d）文字
- **按钮风格**：圆角矩形，带微妙阴影，hover 时金色渐变
- **字体**：标题使用 Playfair Display（衬线体，传达典雅），正文使用 Source Han Sans / Noto Sans SC
- **布局风格**：左侧导航 + 右侧内容区，卡片式信息展示
- **图标风格**：线性图标，金色描边

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 仪表盘 | 资产概览卡片 | 四宫格卡片，数字动画，金色图标，渐变背景 |
| 仪表盘 | 饼图区域 | 环形图，金色/绿色配色，hover 显示详情 |
| 仪表盘 | 近期变动 | 时间线列表，左侧图标标识类型 |
| 资产录入 | 表单区域 | 分步表单，类别选择卡片，输入框带标签 |
| 资产录入 | 列表区域 | 表格+卡片混合，支持搜索筛选 |
| 统计分析 | 图表区域 | 多种图表类型，支持切换视图 |

### 4.3 响应式

- 桌面端优先设计
- 支持 1024px 以上屏幕
- 移动端自适应（导航折叠为汉堡菜单，卡片单列排列）

### 4.4 3D 场景指导

不适用（本工具为数据管理型应用，无需 3D 场景）
